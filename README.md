Meet **ScaleX**, a project management tool to track issues, run cycles, and manage product roadmaps without the chaos of managing the tool itself.

## 🚀 Installation

Getting started with ScaleX is simple. Choose the setup that works best for you:

- **Self-host ScaleX** — Run ScaleX on your own infrastructure for full control over your data.

| Installation methods | Notes                                  |
| -------------------- | -------------------------------------- |
| Docker               | Use the bundled `docker-compose.yml`.  |
| Kubernetes           | Deploy via your preferred Helm setup.  |

`Instance admins` can configure instance settings from the admin dashboard.

## 🌟 Features

- **Work Items** — Create and manage tasks with a robust rich-text editor that supports file uploads, sub-properties, and cross-referenced issues.
- **Cycles** — Track progress with burn-down charts and keep your team's momentum on schedule.
- **Modules** — Break complex projects into smaller, manageable pieces.
- **Views** — Build custom filters to surface only the most relevant issues. Save and share them with your team.
- **Pages** — Capture and organize ideas with AI assistance and a rich-text editor. Format text, add media, insert hyperlinks, or turn notes into actionable items.
- **Analytics** — Real-time insights across your data. Visualize trends, remove blockers, and keep projects moving.
